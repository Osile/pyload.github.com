Windows

The windows build has no dependencies and is portable. Just extract and start.

Download pyload-05.exe

Ubuntu / Debian

Just install via Packagemanager or with command-line:

{% highlight bash %} dpkg -i pyload-v0.4.9-all.deb {% endhighlight %} Download pyload-05.deb

Archlinux

There is a pyLoad package located at the AUR. We suggest installing using yaourt.

{% highlight bash %} yaourt -S pyload {% endhighlight %}

iPKG

If you own a router or NAS with IPKG installed you can use this package. pyLoad will be installed under /opt, please make sure that this is your IPKG installation folder.

{% highlight bash %} ipkg install wget tesseract-ocr tesseract-ocr-lang-eng ossp-js unrar wget http://get.pyload.org/static/pyload-v0.4.9-noarch.ipk ipkg install pyload-v0.4.9-noarch.ipk pyLoadCore -s [For setup, dont copy this message in brackets!] {% endhighlight %}

Source Code
